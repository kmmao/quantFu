# 中国期货合约代码规范

## 概述

本文档详细说明中国期货市场的合约代码标准格式，用于极星量化平台的合约配置。

## 参考地址

https://www.cnopendata.com/data/m/Futures/Futures%2Fhy.html

## 合约代码格式

**标准格式**: `交易所|类型|品种代码|月份代码`

### 示例
- `SHFE|F|RB|2501` - 上海期货交易所螺纹钢2025年1月合约
- `ZCE|Z|TA|601` - 郑州商品交易所PTA601合约
- `DCE|Z|I|2501` - 大连商品交易所铁矿石2025年1月合约
- `CFFEX|Z|IF|2501` - 中金所沪深300期货2025年1月合约

## 主力合约表示

对于主力合约，通常使用以下格式：

```
SHFE|F|RB|MAIN    # 螺纹钢主力合约
ZCE|Z|TA|MAIN     # PTA主力合约
DCE|Z|I|MAIN      # 铁矿石主力合约
CFFEX|Z|IF|MAIN   # 沪深300主力合约
```

## 使用注意事项

1. **交易所代码必须准确**: 使用完整的交易所英文缩写
2. **品种代码区分大小写**: 严格按照官方规范
3. **月份代码格式统一**: 建议使用4位数字年月格式
4. **主力合约动态性**: 主力合约会随时间变化，需要定期更新
5. **合约到期处理**: 注意合约到期时间，及时切换到新合约
